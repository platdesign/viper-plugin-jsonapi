###options

This plugin allows multiple configurations. e.g.

	{
		backendA: {
			path: './backendA',
			baseRoute: '/backendA'
		},
		
		backendB: {
			path: './backendB',
			baseRoute: '/backendB'
		}
	}




####- path *[string]*
Path where the files are located.

####- baseRoute *[string]*
A base route which will prefix every single route of the api.
